## Start

```bash
make
```

## Setup

```bash
make build
```

## Run + Dist

```bash
make run-dist
```

## Run

```bash
make run
```

## Run tests

```bash
make test
```

## Run checkstyle

```bash
make lint
```

## Check update dependencies and plugins

```bash
make update
```
